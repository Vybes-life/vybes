View Examples

0% Discount:

If you set 0% as the buyer discount, the full 30% commission is yours. For a $49 report, you earn $14.70.

15% Discount:

If you select a 15% discount, the buyer receives 15% off, and you earn 15% in commission. For a $49 report, the buyer pays $41.65, and you earn $7.35.

25% Discount:

If you set a 25% discount, you'll earn 5% in commission. For a $49 report, the buyer pays $36.75, and you receive $2.45.

Every 3 days:

Receive smaller, frequent payments, each with a transaction fee. If you have multiple sales, expect a fee on each payment batch.

Weekly:

Receive a weekly payout, reducing the number of transaction fees compared to every 3 days.

Monthly:

Accumulate your commissions for the month and receive a single payout, which minimizes transaction fees since there's only one monthly transfer.
